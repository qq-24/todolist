# 项目意图

## 核心目标
自用极简跨平台 Todo List，通过 GitHub 仓库同步数据，Windows + Android 两端即时可用。

## 已确认的设计决策

### 技术栈
- Flutter + Material Design 3
- 目标平台：Windows、Android

### 数据模型
- 任务字段：标题、描述、截止时间、是否提醒
- 同步载体：GitHub 仓库中的一个 JSON 文档

### 同步机制
- GitHub API 上传/下载，Token 硬编码打包（自用，不做登录流程）
- 混合同步：打开时自动拉取，修改后自动推送，保留手动同步按钮
- 冲突处理：同步前先拉取远端，发现冲突时弹窗让用户选保留哪个版本
- 防重复推送：推送前 hash 比对，无变化不推送

### 交互设计
- 主界面：纯列表，滴答清单风格
- 新建/编辑：底部半屏面板（Bottom Sheet），含标题、描述、日期时间选择器、提醒提前量
- 完成：华丽简约动画 → 变灰沉底
- 删除：左滑，有阻尼感
- 截止时间：MD3 日期+时间选择器
- 提醒：可选提前量（到期时/15分钟/1小时/1天），Android 用 AlarmManager，Windows 用本地通知
- 同步状态：AppBar 图标（转圈/打勾/红叹号）+ 抽屉内详情
- 主题：动态取色（Material You），可设置跟随系统或手动选深浅模式
- 排序：默认添加顺序，可手动拖拽，可切换按截止时间排序
- 新建入口：右下角 FAB
- 侧滑抽屉（Navigation Drawer）：同步详情、手动同步、排序切换、主题设置
- 冲突处理：弹对话框让用户选保留本地或远端版本

### Windows 桌面端特性
- 系统托盘：关闭窗口缩到托盘（首次弹提示可勾不再提示），最小化保持任务栏正常行为
- 托盘图标三态：正常/同步中（蓝色）/同步失败（红色）
- 托盘菜单：打开主窗口 / 手动同步 / 开机自启 ✓ / 退出
- Win11 Toast 通知：标题+内容+"完成"按钮+"稍后提醒"按钮（15分钟后）
- 点击通知：打开主窗口聚焦到对应任务
- 开机自启：注册表方式（便携版兼容），默认开启，侧边栏设置项（仅 Windows）
- 开机自启时带 --minimized 参数直接进托盘不弹窗
- 窗口默认 400×711（9:16），可拖拽调整但强制等比缩放
- 依赖包：tray_manager、local_notifier、launch_at_startup、window_manager

### 明确不做
- 用户登录/注册
- 多用户协作
- Token 输入界面
